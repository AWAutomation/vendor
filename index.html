How to add a new vendor entry
  1. Open this HTML file and find the JSON block marked <script id="records-data" type="application/json">.
  2. Copy one full record object, paste it before the closing ], and give it the next unused numeric id.
  3. Fill in category, subcategory, vendor, contact, phone, mobile, email, website, offerings, and notes. Use an array for any field with more than one value.
  4. Find the JavaScript object named videoCatalog and add a matching key for the same id. Add one video object for each offering: { offering: "Offering Name", id: "YouTubeVideoId", label: "Short title" }.
  5. If the vendor offers Robots as a Service leasing, update raasCatalog: use yes for direct RaaS, third-party for third-party RaaS, or omit the vendor for no RaaS.
  6. Use only the YouTube video id, not the full URL. For example, from https://www.youtube.com/watch?v=abc123, use abc123.
  7. Save the file, refresh the browser, then confirm search, filters, RaaS status, and the video carousel work for the new entry.
